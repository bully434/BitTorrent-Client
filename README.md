# BitTorrent Client
Simple implementation of BitTorrent Client.
### Requirements
```
aiohttp==2.3.10
bitarray==0.8.3
```
###Usage
1. Start a server
    ```
    $ python main.py start
    ```
2. Open another cmd and add .torrent file
    ```
    $ python main.py add (file)
    ```
    Also you can control torrent state:
    ```
   add
   pause
   resume
   remove
   status
    ```
